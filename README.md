# Temperature Anomalies and Cereal Yields

This project analyzes the relationship between temperature anomalies and cereal yields across countries.

## Files
- temperature_cereal_yield_project.ipynb: main analysis
- FAOSTAT_data_en_4-20-2026.csv: cereal yield data
- monthly-temperature-anomalies.csv: temperature data
- requirements.txt: Python dependencies

## How to run
Install dependencies:

pip install -r requirements.txt

Then open the notebook:

jupyter notebook
